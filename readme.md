# Basic Python Firewall

A simple rule-based firewall simulation built using Python.  
The project demonstrates basic firewall concepts such as IP blocking, port blocking, and security logging.

## Features
- Block traffic based on IP address
- Block traffic based on port number
- Rule-based filtering using a configuration file
- Logs allowed and blocked traffic events
- Compatible with Python 3.13

## Technologies Used
- Python 3.13
- Git & GitHub
- VS Code

## How to Run
```bash
py firewall.py
